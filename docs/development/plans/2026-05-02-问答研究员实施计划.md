# 问答研究员实施计划

> **状态：已完成归档。** 本计划对应 README 中的阶段一主体闭环（raw -> ingest -> wiki -> ask），所有 checkbox 任务已在代码基线中实现。当前产品方向已调整为 Dev Run-first，本文件仅保留作为历史参考。

> **给 agentic worker：** 必须使用 `superpowers:subagent-driven-development`（推荐）或 `superpowers:executing-plans` 按任务执行本计划。步骤使用 checkbox（`- [ ]`）语法跟踪。

**目标：** 将 Inkdesk Ask 从轻量 prompt 包装升级为 vault-first 的问答研究员，支持上下文追问、条件式网页辅助，以及可审阅的 `raw -> ingest -> wiki` 写回。

**架构：** 保持单一 Python 主后端和单一 Ask runtime 边界。把 Ask 重构为分阶段 LangGraph 流程，明确包含上下文加载、vault 检索、证据评估、可选网页辅助、回答组合和写回包生成。把 Ask 轮次持久化为可复用研究快照，让前端可以延续追问，并在不重新执行完整研究流程的情况下，把带网页证据的写回物化为审阅提案。

**技术栈：** FastAPI, Pydantic v2, SQLAlchemy 2, LangGraph, httpx, BeautifulSoup, Next.js App Router, Playwright, pytest

def build_topic_citations(self, topic: Topic | None) -> list[Source]:

def build_global_citations(self, question: str) -> list[Source]:
    lowered = question.lower()
