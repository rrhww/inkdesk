# 页面简报

## 1. 公开输出首页

- 目标：让访客一眼理解作者、公开文章和长期项目分享
- 关键词：公开输出、分享层、长期主义、克制
- 必须有：作者介绍、公开文章、项目/链接
- 禁止项：登录入口、产品官网式 CTA、企业产品营销语言

## 2. 登录页

- 目标：只服务主人进入主系统
- 关键词：隐藏入口、克制、可信、私有
- 必须有：品牌信息、登录表单、错误反馈
- 禁止项：把它做成公共入口页

## 3. Agent 控制台

- 目标：成为主人进入主系统后的第一屏
- 关键词：中枢、上下文、推进、协同
- 必须有：Agent 建议、任务摘要、知识入口
- 禁止项：传统 SaaS 仪表盘、企业管理台图表风格

## 4. 笔记与知识库

- 目标：展示系统事实来源
- 关键词：知识资产、沉淀、可追踪
- 必须有：标题、摘要、标签、更新时间、归属状态

## 5. 知识资产编辑器

- 目标：编辑主系统中的笔记与知识资产
- 关键词：专注、发布状态、主系统、输出层
- 必须有：状态标签、正文画布、侧边元信息、发布入口

## 6. 任务与计划

- 目标：让主系统具备推进事情的能力
- 关键词：执行、分层、聚焦
- 必须有：今日任务、进行中计划、状态标记

## 7. 全局检索

- 目标：让历史知识重新进入当前判断
- 关键词：召回、上下文、检索
- 必须有：检索框、结果、无结果说明

## 8. 发布模块

- 目标：把主系统中的成熟内容发布到公开输出层
- 关键词：次级模块、发布、公开输出
- 必须有：已发布内容、待发布内容
- 禁止项：让它看起来像整个产品的中心页面

## 9. 公开文章页

- 目标：稳定承接公开阅读
- 关键词：阅读、作者、回流
- 必须有：标题、正文、作者摘要、返回入口
