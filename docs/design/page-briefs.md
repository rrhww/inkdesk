# 页面简报

## 1. 登录页

- 目标：只服务主人进入私有研究系统
- 关键词：隐藏入口、克制、可信、私有
- 必须有：品牌信息、登录表单、错误反馈
- 禁止项：做成公开主页或营销入口

## 2. 问答

- 目标：成为主人进入系统后的第一屏
- 关键词：研究、上下文、证据、追问
- 必须有：输入区、回答、引用来源、知识缺口、沉淀动作
- 禁止项：传统 SaaS 仪表盘、企业管理台图表风格

## 3. 资料

- 目标：展示和导入原始研究材料
- 关键词：来源、导入、原始、provenance
- 必须有：导入入口、来源列表、状态、vault 规则说明

## 4. 审阅

- 目标：处理 AI 编译提案
- 关键词：审阅、确认闸门、提案、证据
- 必须有：提案摘要、topic 归属、claims、证据、accept / reject

## 5. 知识库

- 目标：展示已经确认沉淀的知识页
- 关键词：长期记忆、沉淀、正式知识、可复用
- 必须有：标题、摘要、来源数量、开放问题数量、详情入口

## 6. 知识页详情

- 目标：承载单个 wiki 页面结构化知识
- 关键词：understanding、claims、questions、sources
- 必须有：当前理解、关键结论、开放问题、来源链接
