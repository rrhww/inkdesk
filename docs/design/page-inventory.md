# 页面清单

## 目标

这份文档逐页说明当前 MVP 需要承载的页面目标、关键模块和状态，作为设计交付与前端实现的页面规格。

## 页面清单

| 页面 | 路由 | 主要目标 | 关键模块 | 关键状态 | 设计优先级 | 首批开发 |
| --- | --- | --- | --- | --- | --- | --- |
| 公开输出首页 | `/` | 展示作者与公开内容 | 作者介绍、公开文章、项目/链接 | 默认、有公开内容 | P0 | 是 |
| 公开文章页 | `/articles/[slug]` | 阅读公开文章 | 标题区、正文区、返回入口、作者信息 | 正常、文章不存在 | P0 | 是 |
| 登录页 | `/login` | 主人进入主系统 | 品牌区、登录表单、错误提示 | 默认、失败 | P0 | 是 |
| Agent 控制台 | `/app` | 主系统第一屏 | Agent 建议、今日上下文、任务摘要、笔记入口 | 默认 | P0 | 是 |
| 笔记与知识库 | `/app/library` | 浏览知识资产 | 列表、标签、更新时间、归属状态 | 默认 | P0 | 是 |
| 知识资产编辑器 | `/app/notes/[id]` | 编辑笔记 | 标题区、正文区、发布状态、公开输出入口 | 空白、加载中、同步中、失败、已发布 | P0 | 是 |
| 任务与计划 | `/app/plans` | 推进事情 | 今日任务、进行中计划、状态标签 | 默认 | P0 | 是 |
| 全局检索 | `/app/search` | 召回上下文 | 搜索框、结果列表、召回提示 | 空状态、有结果、无结果 | P0 | 是 |
| 发布模块 | `/app/publish` | 向公开输出层发布内容 | 已发布列表、仅主系统列表 | 有内容 | P1 | 是 |
| 标签页 | `/app/tags` | 未来按标签组织内容 | 标签列表、标签详情 | 暂缓 | P1 | 否 |
| 设置页 | `/app/settings` | 未来管理主系统配置 | 作者信息、主系统配置 | 暂缓 | P1 | 否 |

## 首批必须出高保真的页面

- 公开输出首页
- 公开文章页
- 登录页
- Agent 控制台
- 知识资产编辑器
- 任务与计划

## 页面状态要求

- 公开输出首页至少覆盖有公开内容状态
- 登录页至少覆盖默认和失败状态
- Agent 控制台至少覆盖默认状态
- 检索页至少覆盖空状态、有结果、无结果
- 编辑器必须覆盖 blank / loading / draft / saving / error / published

## 页面实现说明

- 公开输出首页和公开文章页共享输出层阅读气质
- 主系统页面共享统一私有壳层
- 发布模块继续保留，但不再要求成为主视觉中心
- 标签页和设置页降为未来 P1，不是本轮实现重点
