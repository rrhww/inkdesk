# Stitch 审核清单

## 通用问题

- 这张图有没有明显偏向企业管理台？
- 它有没有让人看不出“超级个人工作台”的气质？
- 公开输出层和主系统有没有被混成一层？
- 页面里有没有出现不该出现的系统入口或营销按钮？

## 公共首页审核

- 看起来像公开输出页和个人分享页，而不是产品官网
- 有作者介绍、公开文章、项目 / 链接
- 没有主系统入口

## Agent 控制台审核

- 第一眼就是 Agent 控制台，而不是普通列表页
- 能看出任务与笔记入口
- 不像企业 dashboard

## 知识资产编辑器审核

- 看起来像长期写作与知识沉淀画布
- 能表达主系统同步状态
- 公开输出发布能力处于次级但可见的位置

## 任务与计划审核

- 真的像主入口之一，而不是附加页
- 能区分今日任务与进行中计划

## 发布模块审核

- 有发布到公开输出层的感觉
- 但不会看起来像整个产品的中心
