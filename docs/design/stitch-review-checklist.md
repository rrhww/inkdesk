# Stitch 审核清单

## 通用问题

- 这张图有没有明显偏向企业管理台？
- 它有没有让人看不出“私人研究工作台”的气质？
- 用户能不能一眼分清 `资料 / 审阅 / 知识库` 三层？
- 页面里有没有出现旧路线残留？

## 登录页审核

- 看起来像私有入口，而不是公开主页
- 没有营销感

## 问答审核

- 第一眼就是研究问答入口，而不是普通列表页
- 能看出来源、知识缺口和沉淀动作
- 不像企业 dashboard

## 资料审核

- 看起来像原始材料工作区
- 导入入口足够明显
- 能理解这些内容还没有变成正式知识

## 审阅审核

- 能一眼看出这是 AI 提案确认闸门
- 能看懂 accept / reject 对知识层的影响

## 知识库审核

- 看起来像正式知识层，而不是普通文件列表
- 能表达“沉淀后内容”的稳定感
