# HTML 方向复核

## 结论

旧方向中的分叉心智已经不再适合当前产品。

现在的评审标准改为：

- `问答` 必须像可信的研究入口
- `资料` 必须像原始证据层
- `审阅` 必须像人工确认闸门
- `知识库` 必须像正式沉淀层

## 当前明确要求

### 问答必须像什么

- 研究入口
- 可继续追问的连续对话
- 带来源和知识缺口的回答环境

### 问答不能像什么

- 通用聊天玩具
- 企业 dashboard

### 资料必须像什么

- 原始材料仓
- 导入与来源追踪入口

### 审阅必须像什么

- AI 提案确认区
- 通往 wiki 的人工闸门

### 知识库必须像什么

- 已确认知识页集合
- 可长期复用的研究记忆

## 后续动作

- 所有 HTML / Stitch 方向都按 `raw -> ingest -> wiki -> ask` 复核
- 不再沿用旧路线叙事
